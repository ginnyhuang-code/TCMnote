SLE・系統性紅斑狼瘡

最常用藥物頻率

核心病機

腎虛為本，熱毒為標，瘀血濕濁為關鍵病理。自體免疫攻擊肝血 → 滋陰清熱為主軸。

神經症狀

牡丹皮＋白芍護肝血；四肢麻木加知柏地黃丸；心跳快加甘露飲。

晚發型（50歲後）

男女比1:1，較輕微。人衰老：肝腎陰虛 → 脾腎陽虛，可加虎潛丸、歸脾湯。

氣虛型（肥胖）

補中益氣湯為主；舌診儀顯示三焦有濕、氣虛血瘀時參考。

衛教重點

禁烤炸辣、苜蓿芽、補品；不熬夜；避免光敏感。

皮疹

消風散＋甘露飲＋牡丹皮＋薏苡仁；光敏感性紅疹可加板藍根。

乾燥症

最常用藥物頻率

核心病機

陰虛為本，瘀痺燥為標。天癸漸竭，衝任虛損，燥邪乘虛而入；或風寒濕入裡化燥傷陰。

治則

急性期（發炎）：清營湯（銀翹連葉犀增丹）。慢慢加入活絡滋陰益氣；酸甘化陰防滋膩。

大便乾結

腸液津虧 → 加太子參健脾益氣，加強腺體分泌。

失眠

酸棗仁湯最常用；打疫苗後眠差亦可用。

問診要點

發病前狀態、素食？反覆感冒（伏邪）？睡眠？經帶胎產史（耗陰化火）。

間質性腎炎合併

以西醫為主（奎寧、類固醇、leflunomide），中醫輔助減少發炎。

RA・類風濕性關節炎

最常用藥物頻率

底層病機

與SLE、乾燥症同源，陰虛為本，故知柏地黃丸出現率最高（3/3則）。

關節變形、晨僵

桂枝芍藥知母湯＋虎潛丸；泡熱水緩解、冬天加重 → 寒濕痹。

濕熱明顯

消風散、涼膈散、龍膽瀉肝湯；便秘 → 調胃承氣湯。

纖維肌痛症合併

當歸捻痛湯＋知柏地黃丸＋香砂六君子湯護胃；衛教：少量多餐，備蘇打餅。

外用輔助

手指按摩器；魔鬼爪根精油（非洲植物）。

纖維肌痛症

痛部位會移動、個性龜毛緊張；抑肝散＋牡丹皮；針灸：曲池、手三里、外關。

眼科

最常用藥物頻率

飛蚊症

急性（閃光）→ 滋陰降火湯；慢性 → 滋腎明目湯；最常用：杞菊地黃丸。

眼底血管

動脈阻塞（眼中風）→ 通竅活血湯；靜脈阻塞（火焰狀出血）→ 血府逐瘀湯；氣虛出血 → 歸脾湯。

黃斑部病變（乾性）

杞菊地黃丸＋桂枝茯苓丸＋養肝腎活血化瘀；引經藥用防風。

女性眼疾

一定調月經，肝腎同源，當歸白芍最重要；四物五子丸治經行目澀。

引經藥

防風最常用；羌活、白芷、蒿本 → 引藥上達眼目。右眼氣、左眼血。

鼻過敏引起眼過敏

優先處理鼻過敏（荊防敗毒散＋桑菊飲＋辛夷清肺湯），眼症自然改善。

腎病

最常用藥物頻率

核心思路

處理underlying disease為主；利水一點即可，引藥到腎，過多利水反不好。

分型用藥

心臟差 → 濟生腎氣丸；代謝型三高 → 知柏地黃丸；心肺腎合併 → 濟生腎氣丸＋生脈散＋炙甘草湯。

減少發炎纖維化

連翹＋丹參（減少腎發炎 → 纖維化 → 萎縮的進程）。

養陰注意

養陰須有陰虛表現（舌紅脈數少苔）；若苔膩先清濕熱，用知柏地黃丸。

消化系統

最常用藥物頻率

便秘

手術傷神經 → 潤法（仁劑）；老人大腸無力 → 補氣（太子參、西洋參、黃芪）；滋陰 → 肺陰胃陰。

腸胃炎腹瀉

熱瀉 → 葛芩連湯；脾虛 → 參苓白朮散；噁心 → 半夏＋紫蘇葉。

腹脹

大柴胡湯推動大腸；半夏厚朴湯降逆；砂仁＋紫蘇梗疏氣。

胃悶痛

香砂六君子湯＋半夏瀉心湯；白及護胃黏膜；麥芽神麴消食開胃。

神經系統

最常用藥物頻率

周邊神經病變

桂枝茯苓丸促循環＋天麻鉤藤針對麻木；最可能病因：DM、化學毒物。

眩暈

苓桂朮甘湯增加內耳循環；良性陣發性姿勢性眩暈（耳石脫落）最常見。

記憶力衰退

黃芪黨參補氣＋石菖蒲遠志開竅＋肉蓯蓉補腎；衛教：記憶需不斷訓練形成迴路。

纖維肌痛症

抑肝散＋牡丹皮生地；痛部位移動、個性緊張為特點；針灸：陰陵泉、陽陵泉、委中。

皮膚系統

最常用藥物頻率

血虛風燥（皮膚乾癢）

當歸飲子：四物＋祛風益氣；熟地改生地涼血；何首烏養血潤燥。

濕熱皮疹

消風散＋甘露飲＋薏苡仁；足癬：本質虛，常見濕熱。

見熱象不必急清熱

產後易汗夾熱：當歸補血湯（血虛生熱）；小建中湯治手足煩熱、咽乾口燥。

出血辨證

脈有力（熱）→ 側柏葉荊芥炭；肺咳血 → 仙鶴草；血尿 → 涼血；月經後期點滴不淨 → 黃芪。

其他

最常用藥物頻率

高血壓

天麻鉤藤飲＋白虎湯＋龍膽瀉肝湯；丹參白芍活血。

痛風

急性：四妙散＋銀翹＋桃紅＋金錢草。間歇期：化痰瘀＋黃芪。外用：如意金黃散。

睡眠呼吸中止症

散腫潰堅湯（痰核）＋大柴胡湯（肥胖）；衛教：多游泳（雙側交替動作）。

AS 強直性脊椎炎

天王補心丹＋柴胡桂枝湯；衛教：多游泳（雙側交替動作較佳）。
