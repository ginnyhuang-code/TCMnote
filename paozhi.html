炮製學筆記

目錄

常用輔料

分為液體輔料與固體輔料：

液體輔料

  • 酒（酒炙黃芩）
  • 醋（醋炙延胡索）
  • 鹽水（鹽炙澤瀉）
  • 薑汁（薑炙厚朴）
  • 蜜（蜜炙甘草）
  • 米泔水（米泔水炒蒼朮）
  • 甘草汁（甘草湯製遠志）
  • 吳茱萸汁（吳茱萸製黃連）

固體輔料

  • 砂、土、麥麩、豆腐

炮製對理化性質的影響

生物鹼

  • 酒製：提高生物鹼溶出率
  • 醋製：提高水溶液中溶出量（醋製延胡索，增加止痛）

含苷類（配糖體 glycoside）

大黃（anthraquinonli 瀉下）、甘草（三萜類 glycyrrhizin）

  • 易溶於水，為避免水解宜少泡多潤
  • 易被酶分解，使有效成分減少（黃芩苷 baicaline 及漢黃芩苷 woogonoside 被水解→綠黃芩，故以熱水炮製（燙製）是為了破壞酶，防止水解）
  • 切製好的黃芩，其 baicaline 含量：蒸 ＞ 煮 ＞ 冷水浸
  • 易被酸水解

揮發油

  • 及時處理藥材，以免揮發油損失（加水宜搶水洗）

鞣質（tannin）

  • 含多酚基，易溶於水
  • 耐高溫
  • 有收斂作用
  • 大黃含有 anthraquinonline 瀉下 + tannin 收斂→經加熱可使致瀉作用減弱，收斂作用相對增加
  • 與鐵形成墨綠色化合物，應避免

有機酸

  • 山楂炒後可破壞部分有機酸，降低胃腸道刺激
  • 常用甘草水（含甘草酸）製含生物鹼藥物，增強療效

油脂類

  • 潤腸通便、致瀉
  • 加熱、壓榨去油以降低毒性（巴豆製霜）、消除副作用

樹脂類

  • 酒、醋製提高溶解度

蛋白質 / 醣類

  • 少用水處理，宜少泡多潤

無機化合物

  • 煅燒（煅明礬）、煅紅醋淬

炮製與臨床功效

淨製

  • 去除雜質、非藥用部位（巴戟天的木心應去除）
  • 去除混有外形相似的其他有毒藥物（八角茴香混入莽草）

切製

  • 桂枝以氣勝，白芍以味勝（均切薄片，才可達氣味共存的目的）

炒製

  • 炒焦（炒麥芽）：啟脾開胃
  • 炒梔子：緩和苦寒之性
  • 麩炒殭蠶：矯臭
  • 荊芥：生用發汗解表，炒炭止血
  • 炮薑：溫中散寒
  • 川烏加熱後，降低毒性（烏頭鹼）

輔料製

  • 酒製（辛熱行散）苦寒藥：緩和苦寒
  • 酒製活血藥：作用增強、力速
  • 醋製活血藥：緩、久（醋五靈脂）
  • 鹽製（味）溫腎藥（氣厚）：氣味相扶
  • 薑製：增強化痰止嘔（薑半夏）
  • 蜜製（甘溫益氣）：增強止咳或補氣藥

藥汁製

  • 吳茱萸（辛熱）製黃連（苦寒）：一冷一熱，陰陽相濟，無偏勝之害。瀉肝火以和胃氣（左金丸）
  • 甘草汁製遠志：
    • 遠志祛痰作用（含 tenuigenin 遠志皂苷），易溶於熱水。甘草鎮咳袪痰（含 glycyrrhizin 甘草甜素）。熱甘草水泡遠志，起協同作用，增強鎮咳袪痰作用
    • glycyrrhizin 水解成 glucuronic acid 具解毒作用，可減輕副作用

淨選

1. 清除雜質、黴變品

  • 挑選
  • 篩選：藥物和雜質體積大小不同
  • 風選：藥物和雜質比重不同
  • 水選（水洗、漂）去除泥沙、鹽分

2. 進行分檔（長短大小厚薄）

雷公炮製17法的「度」，避免個小太過、個大不及

3. 去除非藥用部位

  • 去根、莖
  • 去皮殼：去栓皮（杜仲、厚朴）、根皮（知母、桔梗）、果殼（白果）、種皮（苦杏仁）
  • 去毛（刮、刷、燙、挖、撞）：避免對咽喉刺激，引起咳嗽
  • 去心：蓮子（去種子胚芽）、遠志（去藥材木質部）——去心者免煩
  • 去蘆（蘆頭）：人參、黨參（參蘆為湧吐劑）——去蘆頭者免吐
  • 去核：花椒、山茱萸——去核者免滑
  • 去瓤：枳殼——去瓤者免脹
  • 去枝梗
  • 去頭尾足翅（昆蟲）
  • 去殘肉（龜板、鱉甲）

4. 分開藥用部位

切製

切製前的水處理

  • 淋法：避免返熱爛葉，切後應即時乾燥
  • 淘洗法（搶水洗）：快速洗滌，縮短藥材與水接觸時間，防止藥材「傷水」和有效成分流失。大多洗一次即可
  • 泡法：少泡多潤原則
  • 漂法：多量水、多次清洗。除去有毒物成分（川烏、天南星）、鹽份、腥臭味
  • 潤法：保持濕潤狀態，內外濕度一致。分為浸潤、伏潤（悶潤）、露潤（吸濕回潤）

飲片切製目的

  • 便於有效成分煎出（質堅宜薄、質鬆宜厚）
  • 提高煎藥質量，並可避免藥材細粉糊化、黏鍋（飲片細而不粉）
  • 利於炮製
  • 利於調配、儲存
  • 便於鑑別
  • 利於製備

飲片類型

  • 蝴蝶片：川芎
  • 頂頭片（圓片、橫片）：熟地黃、當歸
  • 斜片（懷牛膝）：又分馬蹄片（黃耆）、柳葉片、瓜子片
  • 直片（順片）：當歸
  • 絲狀片：細絲（質地硬者切細）厚朴、寬絲（質地軟者切寬）
  • 盤香片：厚朴
  • 如意片
  • 段：黨參、懷牛膝
  • 丁：正立方體

炙法

定義：使液體輔料（常用酒、醋、鹽水、薑汁、蜜）滲入藥物組織內部的炮製方法。加入定量液體輔料，入鍋拌炒至藥材乾燥。

炙法與加輔料炒的比較

  • 炙法：液體輔料、溫度較低（文火）、翻炒時間稍長
  • 加輔料炒：固體輔料、溫度較高（中火）、翻炒時間較短、固體輔料作為導熱介質

酒製（米酒）

具活血通絡、袪風散寒、矯臭去腥作用

目的

  • 提高有效成分溶出率（如生物鹼）
  • 增強活血通絡作用（當歸、川芎）
  • 矯臭去腥（烏梢蛇等動物類藥物）
  • 改變藥性，引藥上行：大黃、黃連、黃柏苦寒沉降下行，多用於清中下焦濕熱，酒製後可清上焦實熱

方法

  • 先拌酒，後炒藥（質地堅硬的根、根莖類）
  • 先炒藥，後拌酒（質地疏鬆，如五靈脂）

例子

  • 酒製黃芩：增加有效成分浸出、抑制苦寒之性，加強上行
  • 酒製白芍：可緩和酸寒之性、增加芍藥苷 paeoniflorin 在水中溶解度而增加通經止痛效果
  • 酒製大黃：緩和苦寒瀉下，引藥上行以清上焦實熱（抵當湯、瀉心湯）。SennosideA,B（雙聚物，具有瀉下作用）→經炮製後降解成 rhein 及 emodin（游離型，無瀉下作用）

比較：黃芩的不同炮製

  • 燙製：破壞酶，防止黃芩苷 baicaline 及漢黃芩苷 woogonoside 被水解
  • 酒製：增加有效成分浸出、抑制苦寒之性，加強上行

比較：白芍的不同炮製

  • 酒製：可緩和酸寒之性、增加芍藥苷 paeoniflorin 在水中溶解度而增加通經止痛效果
  • 炒製：破壞安息香酸（苯甲酸），降低對胃腸道刺激

醋製

入肝經血分，具收斂、解毒、散瘀止痛、矯味作用

目的

  • 引藥入肝，增強活血止痛（乳香、三稜、延胡索、五靈脂）、疏肝行氣（柴胡、香附）
  • 降低毒性，減少副作用，適用於峻下逐水藥（大戟、甘遂、芫花）
  • 矯味矯臭（五靈脂）

方法

  • 先拌醋，後炒藥
  • 先炒藥，後拌醋：樹脂類、動物糞便（若先拌醋，會黏成碎塊）

例子

  • 醋製延胡索：提升止痛有效成分生物鹼溶出（醋酸鹽）、增強行氣止痛（醋炒＞醋煮＞醋拌）
  • 醋製香附

比較：香附的不同炮製

  • 醋製：入肝經，增強疏肝止痛
  • 酒製：通經脈，散結滯，治寒疝腹痛
  • 四制（酒醋鹽薑）：行氣解鬱、調經散結
  • 香附炭：止血，治婦女崩漏

鹽製

性味鹹寒，有清熱涼血、軟堅散結、潤燥作用

目的

  • 引藥下行，增強療效
  • 增強滋陰降火（知母、黃柏）
  • 緩和藥物辛燥（補骨脂、益智仁）

方法

  • 先拌鹽水後炒
  • 先炒後噴鹽水：含黏液質多（車前子、知母）遇水易發黏，鹽水不易滲入

例子

  • 鹽製杜仲（砂炒後噴鹽水或浸2%鹽水後烘烤）：鹽炒可直走下焦，增強補肝腎作用。生杜仲含杜仲膠，將有效成分包住，經炒斷絲（橫斷有絲、縱斷無絲）後，膠質被破壞，有效成分易煎出。步驟：去栓皮→切塊→翻炒砂子製砂熱→將杜仲放入鍋中翻炒，至藥材內面起冒，斷絲即可→取出過篩→噴少許食鹽水，攤開放涼
  • 鹽製澤瀉（五苓散中，以澤瀉為使，鹹味泄腎氣）

薑製

生薑含揮發油（增加血液循環）、薑辣素（刺激胃液分泌，促消化功能）。辛溫能散寒解表、溫中止嘔、袪痰止咳、解毒。

目的

  • 薑製溫散而開痰：緩和寒性、增強溫中止嘔
  • 緩和副作用（厚朴有刺激咽部副作用），增強療效
  • 降低毒性（實際上減毒關鍵是因為加熱煮過或炒乾而減毒，非薑本身作用）：薑半夏、薑製天南星（天南星科植物有黏膜刺激性、刺舌頭毒性，來自於草酸鈣針晶體）

例子

  • 薑製厚朴

蜜製

蜂蜜生則性涼，熟（煉蜜）則性溫（能補中）、能解毒、潤燥、止痛、矯臭矯味

目的

  • 增強潤肺止咳（枇杷葉、百合）：蜜製甘緩而潤肺
  • 增強補中益氣（甘草、黃耆）
  • 緩和藥性（麻黃：緩和發汗解表，增強平喘止咳）
  • 矯味矯臭

例子

  • 蜜製甘草：甘草生品甘涼，瀉火解毒，化痰止咳。蜜製甘草甘溫，補脾和胃。甘草甜素 glycyrrhizin（有皮質醇活性）→水解成甘草次酸 glycyrrhetinic acid + 葡萄醣醛酸 glucuronic acid。蜜製甘草後，皮質醇活性減弱，降低副作用
  • 蜜製黃耆
  • 蜜製麻黃
  • 蜜製枇杷葉：生品清肺止咳，蜜製後潤肺止咳

炒製

清炒

炒黃（文火）

  • 決明子炒黃（炒到種皮破裂）：生用滑腸（恩菎類），炒後緩和藥性，種皮破裂助於有效成分煎出
  • 萊菔子炒黃：破壞分解苷類的酶

炒焦（中火）：增加健脾消食功效

  • 山楂炒焦：生用善消食，活血化瘀。炒焦後酸味減弱，緩和藥性
  • 梔子炒焦：炒後緩和苦寒之性，緩和其清熱涼血（梔子素、梔子苷）作用

炒炭（武火）：增強止血功能、降低烈性，但需燒存性

  • 地榆炒炭
  • 茜草炒炭
  • 荊芥炭

加輔料炒（多中火或武火）

  • 砂炒乾薑（炮薑：炒炭，要存性）
  • 麩炒：和中養胃、賦色、焦香促進食欲
    • 麩炒枳實：緩和藥性（去其揮發油、去其苦酸味）、防止橙皮苷 hesperidin 酶解
    • 麩炒枳殼
  • 米炒班蝥：降低毒性（班蝥素）、去腥臭
  • 土炒白朮：緩和藥性（揮發油損失，對胃腸刺激減少）、土炒後增強健脾和胃（黃土炒後成鹼性氧化物，可中和胃酸）
  • 米泔水炒蒼朮：緩和藥性（去油、緩其燥性）
  • 砂燙杜仲（斷絲）：增強療效（破壞膠質、有效成分溶出）
  • 砂燙馬錢子：炒至黃棕色鼓起、十字裂紋，砸開內部紅褐色。馬錢子主成分為番木龞鹼（strychnine），溫度不可過高以免有效成分被破壞
  • 蛤粉炒阿膠：火力較弱，傳熱慢，適合炒製膠類藥物
  • 滑石粉炒：傳熱慢，適合炒韌性大的動物類藥（水蛭、象皮）

煅製

定義：於耐火容器或直接於火上用大火燒煅

目的：改變藥材結構或成分

功效

  • 使質地疏鬆，易於粉碎或吸收（煅者去堅性）：煅瓦楞子（碳酸鈣→氧化鈣，增強制酸作用）
  • 去除結晶水，增強收斂效果：煅明礬（成分為含水硫酸鋁鉀，煅後稱枯礬：硫酸鋁鉀）
  • 緩和藥性

適合：質地堅硬（龍骨、牡蠣）、含結晶水（明礬）

方法

明煅法（直火煅法）：不隔絕空氣

  • 直接火煅：適用煅燒後不會破碎的礦石（石膏）
  • 鐵鍋煅
  • 坩堝煅：嘟嚕罐是耐火土燒成的坩堝（藥料質重的用小型嘟嚕、藥料質輕的用大型嘟嚕）

燜煅法（扣鍋煅法）：密閉、缺氧條件下燒至黑色、存性

  • 鐵鍋燜煅
  • 灰火燜煅：無火焰
  • 爐火燜煅（煅牡蠣）

淬製・煮製・蒸製

淬製

定義：由煅或砂燙後之高溫，趁熱投入冷的液體（醋或酒）中，使其急速降溫

目的

  • 使晶格間完全裂解，使結構酥脆
  • 易於粉碎，利有效成分煎出
  • 清除雜質、去腥味
  • 醋淬磁石：醋酸鐵
  • 醋淬龜板

煮製

定義：用水或其他液體輔料同煮，也可加入其他藥材共煮。最好用銅鍋，不要用鐵鍋。

目的

  • 消除或降低毒性：清水煮川烏
  • 緩和藥性，增強療效：甘草水煮遠志
  • 清潔藥物：豆腐煮珍珠

例子

  • 甘草湯製遠志
  • 藥炙（甘草、金銀花）川烏：川烏主要毒性成份為烏頭鹼
  • 豆腐煮珍珠：去除表面油膩

蒸製

目的

  • 增加溫補作用，多數是滋補類藥材（地黃、女貞子、五味子）。一般先將藥材用酒潤透後再蒸
  • 改變藥物性能，擴大用藥範圍（生地清熱涼血，蒸製後為滋補作用）
  • 減少副作用（黃精刺激喉嚨、大黃瀉下峻猛）
  • 保存藥效、利於儲存（桑螵蛸）
  • 便於軟化切片（木瓜、天麻）

注意事項

  • 蒸籠下層要放適量清水（三分之二）
  • 一般先用武火，帶圓氣後改為文火。圓氣是指在蒸製過程中，蒸鍋內的水燒開後，水蒸氣在鍋蓋周圍溢散的現象
  • 若蒸太久，藥物上水，難於乾燥（會用鋼杯，避免上水）。上水是指在蒸製過程中，由於連續加熱、蒸製時間過長或處理不當，使藥物吸水過多、難以乾燥的現象

例子

  • 酒蒸地黃：藥性由寒轉溫，藥味由苦轉甜，功能由清轉補
  • 酒蒸山茱萸
  • 酒蒸女貞子：寒滑之性減弱，滋補肝腎作用增強
  • 黑豆汁蒸何首烏：味轉甘厚而性轉溫、消除滑腸腹瀉副作用
  • 清蒸桑螵蛸：消除致瀉副作用、殺死蟲卵、利於保存
  • 黃酒蒸黃精：增強補脾潤肺益腎、去除麻味以免刺激咽喉
  • 黃酒蒸肉蓯蓉：增強補腎助陽
  • 醋蒸、酒蒸五味子：生品歛肺止咳、醋製增強酸澀收斂、酒製增強益腎固精

製霜・製麴

製霜

去油製霜

  • 目的：降低毒性、緩和藥性：巴豆製霜降低毒性、緩和瀉下作用
  • 消除副作用：柏子仁製霜消除滑腸副作用
  • 例子：製杏仁霜：燀法→去油製霜

滲析製霜

  • 目的：製造新藥，增強療效
  • 製西瓜霜：提高芒硝純度，增強清熱瀉火功能

昇華製霜

  • 目的：純淨藥物
  • 製砒霜

煎煮製霜

  • 目的：緩和藥性、擴大藥源
  • 製鹿角霜

風化製霜

  • 製元明粉（＝玄明粉＝風化硝）：無結晶水硫酸鈉（將芒硝風化而得）

製麴

目的

  • 改變原有性能，產生新的治療作用，擴大用藥品種：六神麴、淡豆豉
  • 增強療效：半夏麴（健脾溫胃、燥濕化痰）

步驟

藥料粉碎、和麵粉拌勻→藥汁拌麴→成形→堆麴（排成品字）→發酵（自然發酵）

製六神麴

  • 藥料（杏仁、赤小豆粉末）拌麵粉
  • 藥汁：青蒿、藿香、半夏、赤小豆

燀法・水飛法

燀法

定義：製於沸水中浸煮短暫時間，取出浸於冷水中，分離種皮

目的

  • 保存有效成分並去除非藥用部位：杏仁（破壞 emulsin 以保存苦杏仁苷 amygdalin、去其皮）。HCN 能鎮靜呼吸中樞，達鎮咳平喘
  • 分離不同藥用部位：白扁豆分離出扁豆仁和扁豆衣

例子

  • 杏仁
  • 桃仁
  • 扁豆：生用消暑袪濕、炒用健脾養胃
  • 郁李仁

水飛法

定義：利用粗細粉末在水中懸浮性不同，將不溶於水的礦物或貝殼類藥物精反覆研磨，製成極細膩的粉末

目的

  • 使藥物質地細膩，便於內服外用
  • 去除藥物中可溶於水的毒性物質

例子

  • 水飛珍珠
  • 硃砂水飛（水揮硃砂）：低溫研磨、主成分是硫化汞，去除可溶性汞鹽、砷毒物
